  2. 输入 hexo new page shuoshuo

  3. 你会找到 source/shuoshuo/index.md 这个文件

你可以隐藏一些説説，可配置的参数：

参数 解释
limit 【可选】配置説説显示数量
limit.type 【可选】配置説説显示数量的类型 （date 或者 num）
limit.value 【可选】配置説説显示数量的值 （当 type 配置为 date 时， value 的值必须为日期。当 type 配置为 num 时， value 的值必须为数字，且大于 0
